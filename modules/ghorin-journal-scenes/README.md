# ghorin-journal-scenes

This module allows to open a scene from a link in a Journal by clicking on the link.

## Description
Foundry core feature : When you clic on a link of a scene in a journal, it opens the scene configuration window. It doesn't open the scene.

This module allows to :
- Click on the link : Open the scene
- Control + Click on the link : Open the scene and activate it
- Alt + Click on the link : Open the scene configuration window

## Installation
Install from FoundryVTT setup / Add-on modules / 
   - Search for the module with filter "ghorin journal scene"
   - Install module with Manifest URl : https://gitlab.com/ghorin/ghorin-journal-scenes/-/raw/main/module.json
